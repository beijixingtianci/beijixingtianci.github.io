函数: 更多内容访问

    focus();作用：使某个元素获得焦点。

    blur();作用：移除某个元素的焦点。

    trigger() 是 jQuery 提供的一个方法，用于程序化地触发某个事件。换句话说，它可以模拟用户触发的事件（比如鼠标点击、键盘按键等），并执行绑定到该事件的回调函数。
$(selector).trigger(eventType);  selector：要触发事件的元素选择器。eventType：要触发的事件类型，例如 "click", "mouseover", "keydown"。

    attr() 是 jQuery 中用于获取或设置 HTML 元素属性的函数。
$(selector).attr(attributeName);// 获取属性值
$(selector).attr(attributeName, value);// 设置属性值
$(selector).attr({
attributeName1: value1,
attributeName2: value2
});// 设置多个属性

    on() 是 jQuery 中用于绑定事件监听器的函数。它可以动态地为元素绑定事件，并且支持事件委托，是 jQuery 中处理事件的核心方法之一。
$(selector).on(event, childSelector, data, function);
event：要绑定的事件类型，例如 "click"、"mouseover" 等。支持多个事件类型，用空格分隔，例如 "click mouseover"。
childSelector（可选）：用于事件委托，指定子元素选择器。只有匹配这个选择器的子元素会触发事件。
data（可选）：传递给事件处理函数的额外数据。
function：事件触发时执行的回调函数。

    .val() 是一个方法，用于获取或设置表单元素的值（value）。

    .each() 方法指定为每个匹配的元素运行一个函数。

    .next() 用来获取当前元素的下一个同级兄弟元素（即在 DOM 树中紧跟在当前元素之后的兄弟节点）。

    .eq()用于选择当前选择器匹配元素集合中的某一个特定索引的元素。

    .appendTo();将选定的元素插入到目标元素的内容末尾。
$(selector).appendTo(target);
selector：表示要插入的内容。
target：表示目标元素，selector 会被添加到它的内部末尾。

    .hover()在一个方法里同时处理鼠标进入和离开的逻辑。
$(selector).hover(handlerIn, handlerOut);
selector：目标元素的选择器。
handlerIn：当鼠标进入目标元素时执行的函数。
handlerOut：当鼠标移出目标元素时执行的函数。
