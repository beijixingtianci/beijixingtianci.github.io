分三个方式插入js：

1. 内部： 在<head>或者<body>里面直接插入<script>

2. 外部： 在<head>里面加上 <script src="filename"></script>

3. 内联： 直接在原tap里面添加

4. 基本类型没有属性和方法，但 JavaScript会自动用包装对象让你访问它们的属性和方法。包装对象是临时的，使用完方法后会自动销毁。手动创建包装对象可能会引发意外问题，不建议手动包装。

5. == 会自动进行类型转换，比较值的“内容是否相等”。=== 不进行类型转换，要求类型和值完全一致，更加严格和安全。

6. 对象比较的是引用，而不是内容。即使两个对象内容完全相同，它们是不同的对象，引用不同，所以被认为不相等。


js函数说明:

    在 JavaScript 中，只调用函数名是对函数的引用,绑定的只是函数的名称，而不是立即执行函数。而函数名＋()就是调用函数的结果,如果函数没有返回值，啥都不是

    document.write(); 写入内容，放在函数里会让之前写的内容消失

    alert(message);弹出警告窗

    comfirm(message);是一个内置方法，用于显示一个带有消息和 "确定" (OK) 和 "取消" (Cancel)按钮的弹窗。它通常用于向用户确认某个操作。

    prompt(message);用于显示一个带有输入框的对话框，用户可以通过输入框输入内容，返回值是用户输入的字符串。

    setTimeout(function, delay);内置的 JavaScript 定时器函数function：要延迟执行的函数。delay：延迟时间，单位是毫秒

    .innerHTML;可以用来 获取或设置 HTML 元素的内容

    window.onload;用于定义当整个页面（包括所有资源如图片、脚本、CSS 文件等）,加载完成后要执行的代码。当页面加载完成时，会触发 load 事件

    parseInt(待解析的字符串,（例如：10 表示十进制）); 是用于将字符串解析为整数的工具函数。

    document.createElement(tagName) 用于动态创建 HTML 元素的方法。

    element.setAttribute(attributeName, value); 用于设置 HTML 元素属性的一个方法。

    parentNode.appendChild(childNode); 将新创建的元素添加到父元素中。

    parentNode.removeChild(childNode); 通过父节点调用，用来删除指定的子节点。

    setInterval(function, milliseconds, param1, param2, ...); 按照指定的时间间隔（以毫秒为单位）重复执行某个函数或代码块。

    clearInterval(intervalID); intervalID：这是由 setInterval() 方法返回的 ID，表示要停止的定时器。

    Math.random();用于生成随机数的一个方法，它会返回一个 0（包含）到 1（不包含）之间的随机小数。

    Math.floor();向下取整，将小数变为整数。

    string.charAt(index);用于获取字符串中某个位置字符的方法。

    string.indexOf(searchValue);用于查找字符串中某个字符或子字符串的位置的方法,返回索引。
